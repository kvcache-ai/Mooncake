# Integrations

Connect Mooncake to LLM serving frameworks and cache systems.

| Integration | Description |
|-------------|-------------|
| [SGLang](sglang/index) | PD disaggregation through Transfer Engine and HiCache L3 storage through Mooncake Store. |
| [vLLM](vllm/index) | Disaggregated prefill/decode and shared KV cache storage. |
| [LMCache](lmcache) | Use Mooncake as a distributed storage backend for LMCache. |
| [LMDeploy](lmdeploy) | Configure Mooncake as the PD disaggregation backend for LMDeploy. |
